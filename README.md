drama meter pro max
calculate the exact amount of drama in your life right now
what it does
pick who the drama is with, describe the situation, and hit calculate. gives you a drama score out of 100 with a color coded progress bar (green = chill, orange = mid, red = chaos) and random advice. every entry gets logged in a history table below.
how to use

select a category (friend, partner, coworker, family)
type out the situation
click calculate drama level
cry or dont

tech
just plain html css and js. no libraries no frameworks nothing. open the file in a browser and it works
